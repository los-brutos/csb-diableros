# Diableros

Le module pour jouer à Diableros avec Foundry et Custom System Builder

## Informations

Ce module implémente le JDR ***Diableros*** sur le **Custom System Builder**.

## Crédits du jeu 

* AEtherya

## Installation du module Diableros

### Installation standard

Dans la liste des Modules, clique sur le bouton Installer pour le module *CSB - Diableros*. Le module s'installera tout seul.

### Installation manuelle

Télécharge l'archive contenant le module. La décompresser dans le répertoire *modules* de Foundry. Redémarre Foundry pour que le nouveau module soit reconnu.

### Configuration

* Crée un monde avec le système Custom System Builder et ouvre-le. 
* Active le module **CSB - Diableros**.
* Importe les templates, les pouvoirs, les supérieur.es et les tables de tirage des sorts qui sont dans les compendiums :
  * Clique droit sur un Compendium pour afficher le menu
  * Clique sur *Importer tout le contenu*
  * Dans la fenêtre qui s'affiche, coche la case *Garder l'ID des documents*
  * Clique sur le bouton *Importer*

Les templates et les objets (spécialités) sont présents dans ton monde et tu peux commencer utiliser le module.
/!\ un template n'est pas une fiche comme les autres : c'est à partir d'un template que vous allez pouvoir créer les fiches des PJ et PNJ

A noter que cette opération d'import sera à faire à chaque mise à jour du module pour bénéficier des mises à jour.
